🏦 Micro_Financial

A lightweight microfinance management system built in PHP

Learn More View on GitHub

💳 Client Management

Register and manage clients with KYC verification, account tracking, and compliance audits.

💰 Loan Operations

Full loan lifecycle: applications, approvals, disbursement tracking, and repayment schedules.

📊 Compliance & Audit

Monitor regulatory compliance, audit trails, and institutional oversight across all operations.

💹 Fund Allocation

Track disbursements, fund allocation, portfolio risk, and collection monitoring in real-time.

👥 Group Lending

Support cooperative loan programs and solidarity-based group lending mechanisms.

📈 Reporting

Generate comprehensive reports, KPIs, and performance dashboards for decision-making.

Tech Stack

PHP 8.x MySQL REST API JavaScript CSS3 Bootstrap

Quick Start

1. Clone the repository:

git clone https://github.com/jhayarfuentes/Micro_Financial.git

2. Set up config.php with your database credentials

3. Import the database schema

4. Run locally or deploy to production
